# Flota 360 — Prototipo Operacional de Taller

Demo ejecutiva para presentar a talleres mecánicos que atienden flotas privadas.

## Cómo correr localmente

Abre `index.html` en cualquier navegador. No requiere servidor ni dependencias.

```
open index.html     # macOS
start index.html    # Windows
```

## Estructura

```
flota360-prototipo-taller/
├── index.html      # App completa
├── styles.css      # Estilos mobile-first
├── app.js          # Navegación e interacciones
└── README.md
```

## Secciones del prototipo

1. Hero — Propuesta de valor
2. Panel ejecutivo — KPIs operacionales
3. Recepción digital
4. Orden de trabajo (con cambio de estado simulado)
5. Diagnóstico técnico (confirmados vs sospechados)
6. Cotización y autorización (con modal)
7. Control de gastos por OT
8. Historial de proveedores (con filtro)
9. Evidencia fotográfica (con tabs)
10. Control de calidad (checklist)
11. Entrega y garantía
12. Portal cliente (tabs por vehículo)
13. Configuración del taller
14. Cierre ejecutivo
